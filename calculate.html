Common DMG (Cdmg):

This dmg considers the basic attack values, and will be applied to both elemental and physical dmg afterwards.

Cdmg = (character base dmg ("P" without anything equipped) + skill atk + tarot + equipment dmg (base + %bonus) + SP attack + enhanced dmg + all atk increased + 15) *
(1 + S% dmg weapon option + S% monster) *
(1 + bonus atk weapon 1 + weapon 2)

Elemental DMG (Edmg):

Everything related to the element applies here. Most important stat being enemy resistance, followed by elemental counter (especially dark/light), the SP and fairy bonus, and the somewhat useful element.

Edmg = [Weapons element (both) + accessories elements (all) + skill element + buffs element + (Cdmg + 100) * (% fairy + SP element bonus)] *
(1 + % Elemental counter) *
(100% - % opponent final resistance)

Physical DMG (Pdmg):

Pdmg = (Cdmg - mob armor) * (1 + % crit dmg - % mob crit reduction)

Morale DMG (Mdmg):

Really useless nowadays.

Mdmg = Player level + Morale buff - Enemy level - Enemy buff

Bonus damage:

Adds the S% from tattoo, Title, costume, atk pot, oil... still testing -> (runes, fairy)

DMG:

Everything sums up or multiplies.

DMG = (Pdmg + Edmg + Mdmg) * (1 + % Bonus damage) * (1 + % final crit dmg)
