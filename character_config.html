Damage calculator

Hi, I'm Dell. I created this tool to help you calculate your damage in the game. It is not 100% finished yet, since some partial buffs need to be included, but most of the latest things are included in the tool. If you have any suggestions, feel free to contact me ingame or contribute to the github repository.

I've iterated the formula a lot of times, but some things need better empirical testing (like the SP wings). Of course, any feedback is welcome, and you can check the formula I use here: Damage formula.
